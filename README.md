# 🚨 TRINETRA – Smart Emergency Response System

TRINETRA is a smart emergency Android application integrated with IoT (ESP32 + MPU6050) that helps users in critical situations by detecting accidents and sending real-time alerts.

---

## 📱 Features

- 👤 Store Personal Details
- 🏥 Medical Information Management
- 📞 Emergency Contacts (up to 4 contacts)
- 🚨 Emergency SOS Button
- 📡 Real-time Firebase Integration
- ⚡ Crash Detection using MPU6050 sensor (ESP32)

---

## 🔧 Tech Stack

- **Android (Kotlin)**
- **Firebase Realtime Database**
- **ESP32 (Arduino IDE)**
- **MPU6050 Sensor**

---

## ⚙️ How It Works

1. MPU6050 detects sudden motion (accident)
2. ESP32 sends data to Firebase
3. App fetches and displays emergency data
4. User can also manually trigger SOS alert

---

## 📂 Project Structure

TRINETRA
